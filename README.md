# Lokesh Portfolio

A modern and responsive developer portfolio built using React and Vite to showcase my projects, skills, certifications, open-source contributions, and coding journey.

## About

This portfolio represents my journey as a Full Stack Developer, AI Engineer, and Open Source Contributor. It highlights my technical expertise, projects, certifications, and problem-solving experience.

## Features

* Responsive and modern UI
* Dark and Light mode support
* Animated Hero section
* Project showcase with modal details
* Resume download option
* GitHub and LinkedIn integration
* LeetCode profile section

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

Go into the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
